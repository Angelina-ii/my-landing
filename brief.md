> Версия 1.0 | Март 2026 | Основан на research.md

---

## 0. СУТЬ ПРОЕКТА

**Что создаём:** Одностраничный лендинг-портфолио цифрового мастера — специалиста, который создаёт сайты, приложения и ИИ-агентов.

**Главная цель сайта:** Получить заявку (написать в Telegram / заполнить форму).

**Вторичная цель:** Сформировать доверие и позиционировать как эксперта уровня "3 в 1" — единственного подрядчика для всего цифрового стека.

**Уникальная фраза проекта:**
> "Один специалист. Сайт. Приложение. AI. Не нужно искать трёх разных подрядчиков."

## 1. ЦЕЛЬ И АУДИТОРИЯ

### Целевая аудитория

| Сегмент | Описание | Главная боль | Что ищет |
|---|---|---|---|
| **Предприниматели МСБ** | Владельцы малого/среднего бизнеса | "Сайт есть — заявок нет" | Работающий инструмент, не витрину |
| **Стартапы** | Команды с идеей, ищущие CTO/разработчика | "Нет технической экспертизы" | Быстрый запуск MVP |
| **Компании-автоматизаторы** | Бизнес, готовый к AI | "Конкуренты уже используют AI, а мы — нет" | ROI, конкретные метрики |
| **Агентства** | Ищут субподрядчика на проекты | "Нужен надёжный исполнитель" | Портфолио, скорость, цена |

### Боли, которые сайт снимает
- Разработчик пропал после предоплаты
- Нужно 3 разных специалиста для 1 проекта
- Непонятно сколько стоит и сколько займёт
- Не знаю, что мне нужно: сайт, приложение или ИИ?
- Уже был плохой опыт с фрилансерами

### Желаемое действие посетителя
1. Написать в Telegram
2. Заполнить форму ("Обсудить проект")
3. Посмотреть конкретный кейс и написать "хочу такое же"

### [10] FAQ

**Задача:** Закрыть возражения до звонка, отфильтровать нецелевые обращения.

**Заголовок:** `Частые вопросы`

**Формат:** Аккордеон (раскрывающийся)

**7 вопросов:**

1. **Как проходит оплата?**
   50% предоплата перед стартом, 50% после сдачи. Работаю по договору, возможна оплата на ИП/самозанятость.

2. **Что нужно от меня для начала работы?**
   Описание задачи или идеи — даже голосовым сообщением. Я сам проведу бриф, сформирую ТЗ и предложение.

   Да, подписываю NDA по запросу. Конфиденциальность клиентов — приоритет.

4. **Что если результат меня не устроит?**
   30 дней бесплатных правок после сдачи. До сдачи — демо каждые 3–5 дней, вы видите и корректируете процесс.

5. **Мои данные в безопасности при работе с ИИ?**
   Да. Использую изолированные окружения, не обучаю модели на данных клиентов. По запросу — развёртывание локальных моделей.

6. **Берёте ли проекты в поддержку после запуска?**
   Да. 30 дней бесплатно, затем — ежемесячный ретейнер от 15 000 ₽/мес в зависимости от объёма.

7. **Как быстро вы отвечаете?**
   В рабочие дни — в течение 2–4 часов. Срочные вопросы — в Telegram.

## 4. СТИЛЬ ДИЗАЙНА

### Общее настроение
**Тёмный. Технологичный. Уверенный.**
Сайт должен ощущаться как продукт из будущего — но тёплый и человечный. Не корпоративно-холодный, не нарочито "хакерский". Эстетика premium tech + личность.

Референсы: Brittany Chiang (элегантность), Eric Van Holtz (уверенность типографики), Aceternity UI (AI-эффекты).

---

### Цветовая система

```css
/* Фоны */
--bg-primary:    #05050F;   /* почти чёрный с синим подтоном */
--bg-secondary:  #0A0A1A;   /* карточки, секции */
--bg-elevated:   #10101F;   /* hover-состояния, модалки */

/* Акцентный цвет (выбрать один) */
--accent:        #7B5EA7;   /* лаванда / фиолетовый — AI-тема */
/* ИЛИ */
--accent:        #00D4FF;   /* электрик циан — tech-тема */
/* ИЛИ */
--accent:        #6EE7B7;   /* мятный неон — свежо */

/* Градиент (для glow-эффектов) */
--gradient:      linear-gradient(135deg, #7B2FBE, #00D4FF);

/* Текст */
--text-primary:  #F0F0F5;   /* основной текст */
--text-secondary:#8888AA;   /* второстепенный */
--text-muted:    #4444666;  /* плейсхолдеры, мелочи */

/* Glassmorphism */
--glass-bg:      rgba(255, 255, 255, 0.04);
--glass-border:  rgba(255, 255, 255, 0.08);
--glass-blur:    backdrop-filter: blur(20px);
```

**Рекомендуемая палитра:** `#05050F` + `#7B5EA7` (фиолетовый акцент) + `#00D4FF` (циан для ИИ-секций)

### Типографика

| Роль | Шрифт | Размер | Начертание |
|---|---|---|---|
| **Заголовки H1** | Clash Display | 72–120px | Semibold / Bold |
| **Заголовки H2** | Clash Display | 40–56px | Medium |
| **Заголовки H3** | Satoshi / Cabinet Grotesk | 24–32px | Medium |
| **Основной текст** | Inter | 16–18px | Regular |
| **Мелкий текст** | Inter | 13–14px | Regular |
| **Акцентные метки** | Satoshi | 12px | Medium, uppercase |
| **Терминальный** | JetBrains Mono | 14px | Regular (для AI-секций) |

**Правила:**
- H1 в hero — занимает значительную часть экрана, доминирует
- Контраст заголовок/текст: крупный Clash Display + мелкий Inter
- Межстрочный интервал основного текста: 1.7

### Принципы верстки

- **Максимальная ширина контента:** 1280px
- **Паддинги секций:** 120px верт. на десктопе, 64px на мобайле
- **Сетка:** CSS Grid 12 колонок + flexbox

- **Скруглення карточек:** border-radius 16–24px
- **Тень карточек:** subtle glow акцентным цветом при hover

- Валидация: React Hook Form + Zod (email, обязательные поля)
